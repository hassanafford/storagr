📋 الخطوة 1: تنفيذ الإعداد الكامل في Supabase (ملف واحد شامل)

1. افتح Supabase Dashboard: https://app.supabase.com
2. اختر مشروعك: xuwoixfgusvufgaliswt
3. اذهب إلى SQL Editor
4. انسخ محتوى ملف: supabase/complete_setup.sql
5. الصق المحتوى في SQL Editor
6. اضغط Run لتنفيذ الإعداد الكامل
7. انتظر حتى تكتمل العملية (قد تستغرق 10-15 ثانية)

⚠️ ملاحظة: هذا الملف يحتوي على:
   ✓ تنظيف كامل للجداول القديمة (إن وجدت)
   ✓ إنشاء جميع الجداول والعلاقات
   ✓ إضافة 12 مخزن + 13 مستخدم
   ✓ إعداد Triggers و Functions
   ✓ تفعيل Row Level Security
   ✓ إنشاء Views مفيدة
